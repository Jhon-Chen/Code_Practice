条件控制语句

{% if user == 'jhon' %} Hello, {{ user }}! {% else %} Hello, Stranger! {% endif %}

for循环语句

    {% for foo in list %}
  • {{ foo }}
    • {% endfor %}

    Jinja2还支持宏，类似于Python中的函数

    为了便于使用，一般将宏放在一个单独的文件夹，使用时用import导入

    需要多出重复使用的模板代码可以写入单独的文件，再包含在所有的模板中，使用include

    {% import 'macros.html' as macros %}
      {% for comment in list %} {{ comment }} {% endfor %}

    另一种重复使用代码的方法是模板继承，它类似于Python中的类继承，比如有一个叫base.html的基模板

    这是base的衍生模板

    {% extends "base.html" %} {% block title %}Index{% endblock %} {% block head %} {{ super() }} {% endblock %} {% block body %}

    Hello World!

    {% endblock %}
